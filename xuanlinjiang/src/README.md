# Source Code

The full source code for this project lives in a separate repository:

**https://github.com/SwiftSeal03/cs2640-proj**

It contains:

- `src/codes/` — erasure code implementations (RAID-0/1/6, RS, MSR, MBR) plus the GF(2⁸) math library
- `src/simulator/` — discrete event simulator, trace translator, in-memory storage array
- `scripts/experiments/` — experiment driver and plotting scripts
- `scripts/traces/` — MSR trace pre-processing
- `test/` — correctness and repair tests

Build and run instructions are in the repo's top-level `README.md`.
