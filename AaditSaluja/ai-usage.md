# AI Usage

I used ChatGPT/Codex as a coding and writing partner while building this
project. I did the project direction and experiment decisions, and I used the AI
to help inspect the repo, write and revise benchmark scripts, debug result
parsing, keep track of run summaries and generate report figures. For important 
design decisions such as the append segment policies, I laid out explicit pseudocode.

Generally, the interactions with the agents was structured and iterative. In each
pass, I would lay out 5-6 bullet points as goals to be achieved. The last was
always to summarize progress and every decision it took in md files. I used them
to better understand every line of code generated but to also have it as context
for a hot start whenever I started a new agent (usually when I ran out of tokens
and would switch between my personal and college accounts). Sometimes, I would also
edit these md files on my own, since they become a medium to communicate. I also 
used it near the end to sync the final CloudLab results and clean up the repository 
layout for submission.

For the report, I generally selected important points from the markdown docs and
asked it to generate a very broad draft. Then I went ahead and wrote the paper.
